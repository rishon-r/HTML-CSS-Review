We can add images to our html pages

            Images are added to the website with the help of the img tag. 
            We use 4 important attributes alongside it.
            Tese attributes are src, height, width and alt.
            src provides the location of image on computer
            height and width are self explanatory and are provide in units of pixels
            alt provides an alternate text to display during times where we cannot display said image
            another good thing about it is when someone is using a screen reader due to visual impairment,
            the screen reader will read out the message provided in the alt attribute
            Note that img is also a self closing tag.

            HTML5 also allows you to add videos to your web pages.
            This can be done using the video tag.
            Height and width tags can be used in a manner similar to how they were used withthe images.
            You use the source tag within the video tag in order to add the link to the video
            You can also use the controls and muted property in the video tag in order to give the user some additional control features
            You can also use the autoplay attribute i oreder to autoplay the video
            You can add a backup link to the video by simply adding another source tag within the video tag
            You also use the type attribute of the source tag in order to state what type of video is being referred to 
            You can also sandwich a video within an anchor tag to make it function as a link similar to how we did it with images.
