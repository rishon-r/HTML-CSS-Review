Lists in HTML

        HTML allows you to create three types of lists.
        They are Ordered Lists, Unordered Lists and Description Lists.
        A description list is a special kind of list that allows  you to have terms and definitions
        HTML also allows you to nest lists within one another.

Example of unordered list: A shopping list

  • Detergent
  • Kitchen Towels
  • Pantry Staples
    • Eggs
    • Bacon
    • Bread
  • Surface cleaner

Example of ordered list: My Favourite Sports

  1. Boxing
  2. Cricket
  3. Football (the European kind obviously)
  4. Basketball
