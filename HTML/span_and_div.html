Span and div tags

 
        An important aspect of html development lies in the understanding of block and inline elements.
        

        Block elements refer to those elements which start on a new line and take up the entire width available
        Block elements can further contain other block and inline elements.
        Examples include div elements, p elements, h1, ul 
        

        Inline elements refer to those elements that do not start on a new line and only take up as much width as needed
        Inline elements usually can only contain other inline elements
        Examples include span,  a, and image
        

        The div tag creates a block container for grouping elements
        This is useful when styling with css
        

        Span wraps inline content without altering the layout
        This means can be placed around small bits of text or inline elements without breaking the flow of the surrounding content.
