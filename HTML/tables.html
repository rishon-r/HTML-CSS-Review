Tables in HTML 


 
        An HTML table is used to organize data into rows and columns just like a spreadsheet.
        An example of a table is given below.
        Tables are created with the table tag. The table tag takes a border attribute which allows us to set the size of the border and it takes an attribute called align.
        A tr tag represents a table row. it takes an align attribute.
        A th tag represents a table header column cell
        A td tag represents a table data column cell
